Amigo Secreto
Manual de utilização

1- Criar a base de dados amigoSecreto. Pode-se utilizar o script amigosecreto.sql para esta finalidade.
2- Criar o usuário para o banco de dados fiap com senha fiap.
3- O login para administração do sistema é admin com senha admin.
4- Pode-se adicionar mais usuários da empresa que queiram participar do amigo secreto na tela de usuario.
5- Pode-se criar grupos distintos (setores) para amigo secreto na tela grupo.
7- Na tela sorteio é possível executar o sorteio para determinado grupo de participantes.
8- Cada participante ao acessar, abre a janela de consulta do amigo secreto sorteado após o sorteio de determinado grupo.
